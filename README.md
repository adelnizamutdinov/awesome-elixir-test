# Awesome Elixir Mirror

## Start

```sh
docker-compose up -d
```

Navigate to http://localhost:4000/

It takes a while for the app to fetch GitHub data for the first time, so you can browse http://104.248.142.159:4000 while it fetches

## Stop

```sh
docker-compose down
```
